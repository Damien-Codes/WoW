# World of Warcraft (WoW) - Text-Based Edition

## 📌 Project Overview

This project is a conceptualized and text-based version of **World of Warcraft (WoW)**, designed for strategic gameplay in a simplified format. It includes a structured **UML** design, battle mechanics, and an inventory system.

---

## 🛠️ Installation and Configuration

To develop and run this project, we are using the following tools:

- **[IntelliJ IDEA](https://www.jetbrains.com/fr-fr/idea/download/?section=windows)** - A powerful IDE for development.
- **[Git](https://git-scm.com/downloads)** - Version control system for managing project updates.
- **[Docker](https://docs.docker.com/desktop/setup/install/windows-install)** - Containerized development and deployment.
- **[StarUML](http://staruml.io)** - UML modeling tool for designing game structure.
